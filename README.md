# 📝 To-Do React App

A simple and interactive **To-Do List Application** built using React.js.
This project demonstrates core React concepts like components, state management, and event handling.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* ⚡ Real-time UI updates using React state
* 📱 Responsive design

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
